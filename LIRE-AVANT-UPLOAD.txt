UPLOAD SUR github.com/bmlemad/EnerTchad — EN UNE FOIS
======================================================

Ce dossier = le site complet et final (165 fichiers). Un seul upload suffit,
il ecrase les fichiers racine deja en ligne et ajoute les dossiers manquants.

LE POINT QUI A FAIT ECHOUER LE 1er ESSAI :
  ne PAS passer par « choose your files » (le selecteur ignore les dossiers).
  Il faut GLISSER-DEPOSER avec la souris, depuis le Finder vers la page GitHub.

ETAPES
  1. github.com/bmlemad/EnerTchad  ->  Add file  ->  Upload files
  2. Dans le Finder, ouvrir ce dossier, Cmd+A (tout selectionner),
     puis GLISSER la selection (fichiers ET dossiers) sur la zone GitHub.
     Attendre la fin de la liste (165 elements, ~1 min).
  3. Commit message : « EnerTchad — release complete 2-3 juillet »  ->  Commit changes.

VERIFICATIONS APRES COMMIT (30 s)
  - Le dossier « amont » apparait dans la liste du depot  -> les dossiers sont passes.
  - « .nojekyll » et « .github » presents ? Si NON (fichiers caches parfois ignores) :
      Add file -> Create new file -> nom « .nojekyll » -> Commit (vide).
      Add file -> Create new file -> nom « .github/workflows/integrity.yml »
        -> coller le contenu du meme fichier de ce dossier (Cmd+Shift+. dans le
           Finder pour voir les fichiers caches) -> Commit.
