# EnerTchad — site web

Société pétrolière intégrée du Tchad. Site statique multipage (8 pôles).
Construit depuis le canon `hydrocarbures.html` via `site-v2/build.py`.

**Mise en ligne** : voir `DEPLOIEMENT-GITHUB-PAGES.md`.

- Accueil + 8 pages-pôles + pages thématiques (Activités, Innovation, Durabilité,
  Entreprise, Investisseurs) + Atlas du secteur + outils interactifs.
- Aucune dépendance serveur : liens relatifs, fonctionne en sous-chemin ou à la racine.
